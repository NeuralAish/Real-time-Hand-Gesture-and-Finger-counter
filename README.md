# ✋ Real-Time Hand Gesture and Finger Counter

## 📌 Description
Real-Time Hand Gesture and Finger Counter is a computer vision–based Python project that uses a webcam to detect human hands and count the number of fingers shown in real time.  
The system supports both left and right hands and can count fingers from 0 to 10 using one or two hands.  
It is built using MediaPipe for accurate hand landmark detection and OpenCV for real-time video processing and visualization.


## 🎯 Features
- Real-time hand detection using webcam  
- Finger counting from 0 to 10  
- Supports single hand and both hands  
- Works with left and right hand  
- Accurate palm and finger landmark tracking  
- Simple and beginner-friendly implementation
